# AI 对口型视频生成技能

## 功能说明
通过上传视频文件与音频文件，AI 自动驱动人物唇形与音频对齐，生成同步对口型视频。
- 需鉴权：APPKEY + 签名
- 计费方式：按最终输出视频时长（分钟）计费
- 渲染方式：云端异步任务

## 鉴权规则
每次提交任务必须携带：
- appkey：平台分配应用密钥
- timestamp：当前时间戳
- sign：MD5 签名

签名生成规则：
```
sign = MD5(appkey + timestamp + secret)
```

## 核心工作流程

```mermaid
flowchart TD
    A[准备视频+音频] --> B[生成鉴权参数 appkey+timestamp+sign]
    B --> C[提交对口型生成任务]
    C --> D{鉴权与余额校验}
    D -->|失败| E[返回错误信息]
    D -->|成功| F[进入AI渲染队列]
    F --> G[异步生成视频]
    G --> H[按实际时长计费]
    H --> I[返回成品视频地址]
```

## 任务提交流程
1. 上传视频、音频至可访问地址
2. 构造鉴权参数并生成签名
3. 调用接口提交任务，获取 task_id
4. 轮询或回调查询任务进度
5. 任务完成后获取结果视频并扣费

## 任务查询内容
- 任务状态：waiting / processing / success / failed
- 处理进度 0~100%
- 实际视频时长（秒）
- 实际扣费金额
- 成品视频下载地址

## 素材要求
- 视频：MP4、MOV，人脸清晰无遮挡
- 音频：MP3、WAV，清晰无杂音
- 时长建议匹配，避免无效时长计费

## 异常说明
- 签名错误/超时：鉴权失败，拒绝任务
- 余额不足：无法提交
- 渲染失败：不计费，可重新提交
```

## 常见问题（FAQ）

### Q：唇形不同步怎么办？
A：
1. 检查音视频时长是否匹配
2. 更换更清晰的正面人脸视频
3. 使用 `--retry` 指令重新渲染

### Q：渲染失败？
A：
1. 检查文件格式与大小
2. 云端渲染检查网络

### Q：导出文件打不开？
A：
1. 检查输出路径是否有效
2. 重新导出或更换格式
3. 使用 `--retry` 重试任务
